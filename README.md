# Bags Technical Assignment

This is the Bags Technical Assignment.


# Assignment Directions

-   Create a react app
-   Use function components
-   Use [Chakra UI](https://chakra-ui.com/) to style the components


## How to run the app:

- To install app, run: 
`npm install`
- To run app, run:
`npm run start`
- To lint app, run:
`npm run lint:fix`
